Sobre o Stater

Starter é um framework de criação de jogos web. O objetivo principal do Starter é suprir o gap entre ferramentas baseadas em linguagem de blocos ou em eventos e as ferramentas profissionais de desenvolvimento de jogos baseanda em linguagem textual.

o Starter oferece os principais recursos para criação de jogos 2D em vários estilos, deixando o desenvolvedor focado na lógica do jogo, através da linguagem javascript.

O Starter possui como referência ferramentas como Scratch e Construct2.

As features atualmente disponíveis são:

  • Classe Bullet (Representa um pojétil)
  • Classe Button (Representa um botão)
  • Classe InputText (representa um campo de entrada de texto por teclado)
  • Classe Text (Representa um texto estático)
  • Classe Rect (Representa um retângulo)
  • Classe Circle (Representa um círculo)
  • Classe Triangle (Representa um triângulo)
  • Classe Background (Permite colocar um background na tela)
  • Carregador de arquivos (Customizável)
  • Controlador de Teclado (Captura as entradas do teclado)
  • Controlador de Armazenamento (Recupera e Armazena informações no navegador)
  • Controlador de Níveis (Controla os níveis do jogo)
  • Classe Caneta (Turtle)

Em desenvolvimento

  • Classe DragDrop (Representa um elemento drogdrop)
  • Classe Platform (Representa um player para jogos em plataforma)
  • Classe Car (Representa um carro)
  • Classe World (Para jogos em plataforma com sistema de colisão)
  • Solid (Representa um Sólido)
  • Muiltdirecionais (Representa um Objeto Multidirecional)
  • Partículas (Representa partículas)

Sugestões podem ser enviados pelo contato de suporte.
