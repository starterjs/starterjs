starterjs

Starter is a framework in javascript for novate in programming. With Starter you can create fun figures and games, manipulate imagens and more. The Starter uses principles founded in major development environment for novice as JES (Jython Environment for Students), Scratch e Construct 2.

See more details in homepage repository or visit the website.
