Sobre o Starter

Starter é um framework para programação contextualizada. O objetivo principal do Starter é suprir o gap entre ferramentas baseadas em linguagem de bloco ou em eventos e as ferramentas profissionais de desenvolvimento, baseadas em linguagem textual.

Tendo como referência ferramentas como JES, Scratch e Construct2, o Starter oferece os principais recursos para criação de Figuras, Efeitos em Imagens, jogos 2D extensível á outros contextos, deixando o desenvolvedor focado na lógica do jogo, através da linguagem javascript.
